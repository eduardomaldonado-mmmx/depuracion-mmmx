Configuración de backend

La plataforma funciona sin backend usando localStorage. Para consolidar los registros de todo el equipo en un Google Sheets, configura un endpoint de Google Apps Script.

1

Crea el Google Sheets destino

Crea un Google Sheets nuevo llamado Depuración MultiMoney. Añade dos pestañas: Depuracion y Expirados. No es necesario poner encabezados, el script los crea automáticamente en la primera captura.

2

Pega el script de Apps Script

Dentro del sheets, abre Extensiones → Apps Script y pega el código del archivo apps-script.gs que te dejé. Guarda con Ctrl+S.

3

Publica como Web App

En Apps Script: Deploy → New deployment → Web app. En "Execute as" elige Me, en "Who has access" elige Anyone (para que los asesores puedan enviar sin login). Copia la URL que te da.

4

Hardcodea el endpoint y define los PINs de los usuarios

Abre index.html en un editor de texto. Busca esta sección al inicio del script:

const CONFIG = {
  BACKEND_URL: 'https://script.google.com/macros/s/.../exec',
};

const USERS = {
  'Manfred':  { role: 'asesor', flow: 'estandar', pin: 'CAMBIA' },
  'Alberto':  { role: 'asesor', flow: 'estandar', pin: 'CAMBIA' },
  ...
  'Eduardo Maldonado': { role: 'admin', pin: 'CAMBIA' },
};

Pega la URL del Apps Script en BACKEND_URL y define un PIN único para cada usuario. Comparte con cada asesor solo su PIN.

5

Publica en GitHub Pages y comparte la URL

Sube index.html a un repo bajo eduardomaldonado-mmmx y activa GitHub Pages. Comparte con todo el equipo la misma URL:

https://eduardomaldonado-mmmx.github.io/depuracion-mmmx/

Cada uno entra con su nombre y su PIN. La plataforma recuerda la sesión en cada navegador hasta que la persona cierre sesión.

Roles

  • Asesor: solo ve la captura de su flujo. Sin dashboard, sin config, no puede cambiar de identidad.
  • Admin: ve todo. Puede cambiar de asesor para probar sin cerrar sesión.

Modo offline: si un asesor captura sin conexión al backend, los registros quedan pendientes en su dispositivo y se envían automáticamente cuando se reconecta.
